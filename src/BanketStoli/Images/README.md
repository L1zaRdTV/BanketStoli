Поместите изображения залов в эту папку. В базе данных можно хранить только имя файла, например `hall1.jpg`; приложение будет искать его в папке `Images` рядом с запускаемым `.exe`.

Также можно указать полный путь к существующему файлу на диске, например `C:\Users\User\Pictures\hall1.jpg`.

Если фото не указано или файл не найден, приложение показывает текстовую заглушку и не пытается открыть несуществующий файл.
